# 04 · 有色雜訊：紅雜訊與 fGn 擴展

Wu & Huang (2004) 的原始理論建構在**純白雜訊**之上。然而現實中許多領域
（氣象、洋流、心電、金融、地震波的低頻段）的背景雜訊都是**有色的**：

- **紅雜訊 / 棕雜訊**：低頻能量高於高頻，有明顯自相關。
- **分數高斯雜訊（fGn）**：以 Hurst 指數 $H$ 量化長程相關性。
  - $H = 0.5$：白雜訊。
  - $H > 0.5$：持續型（長程正相關，如氣候）。
  - $H < 0.5$：反持續型（如湍流某些分量）。

如果誤用白雜訊基準線來檢定有色雜訊背景的訊號，**低頻 IMF 會被錯誤地判定
為顯著**（偽陽性）。

## 為什麼斜率會偏離 −1？

Flandrin et al. (2004, 2005) 延伸了 Wu & Huang 的 dyadic filter bank 分析，
證明對 fGn 而言：

$$\ln(E_m) \approx (\text{斜率}(H)) \cdot \ln(T_m) + c$$

本 repo 採用 Flandrin 的經驗近似：

$$\text{斜率}(H) \approx -2H$$

當 $H = 0.5$ 時斜率回到 $-1$，與 Wu & Huang 一致。

## 實務處理流程

### 步驟 1：估計背景雜訊的 Hurst 指數

常用方法：

- **R/S 分析**（經典但對短資料不穩）。
- **DFA（Detrended Fluctuation Analysis）**（推薦，對非平穩資料較穩）。
- **小波變異數**。

本 repo 尚未內建 Hurst 估計（可由社群貢獻）；在此之前建議使用
[`hurst`](https://pypi.org/project/hurst/) 或 [`nolds`](https://pypi.org/project/nolds/) 套件。

### 步驟 2：修正理論基準線的斜率

```python
from emdsig import red_noise_slope

H = 0.7                              # 來自 DFA 估計
slope = red_noise_slope(H)           # = -1.4
# 基準線：ln(E) = slope * ln(T) + c
```

### 步驟 3：用 Monte Carlo + fGn 產生信賴帶

因為閉式卡方推導仍然綁定 $H = 0.5$ 的假設，**有色雜訊情境下建議改用
Monte Carlo**：

```python
from emdsig import monte_carlo_bounds

ln_T_grid, lo, hi = monte_carlo_bounds(
    N=len(signal),
    n_trials=200,
    alpha=0.05,
    noise="fgn",                     # 關鍵：換成 fGn
    hurst=H,
    emd_method="ceemdan",
    seed=42,
)
```

`monte_carlo_bounds` 內部會呼叫 `emdsig.colored_noise.generate_fgn`
（Davies–Harte circulant embedding 法）來產生高效率、統計正確的 fGn 樣本。

### 步驟 4：繪圖與判讀（與白雜訊情境相同）

```python
from emdsig import plot_significance
plot_significance(ln_T, ln_E,
                  bounds=(ln_T_grid, lo, hi),
                  title=f"EMD 顯著性檢定 (fGn, H={H})")
```

完整示範見 [`notebooks/04_red_noise_extension.ipynb`](../../notebooks/04_red_noise_extension.ipynb)。

## 何時必須關心有色雜訊？

| 應用領域 | 典型背景雜訊 | 建議 Hurst 範圍 |
|---|---|---|
| 氣象、氣候 | 紅雜訊 | $H \in [0.6, 0.9]$ |
| 心電 / 腦波（低頻段） | 1/f 雜訊 | $H \approx 0.5$–$0.8$ |
| 金融報酬 | 近白雜訊 | $H \approx 0.5$ |
| 地震波（微震段） | 紅雜訊 | $H \in [0.6, 0.8]$ |
| 實驗室量測、機械振動 | 常接近白雜訊 | $H \approx 0.5$ |

> **建議**：拿到新資料時，先做一次 DFA 估計 $H$。若 $H$ 顯著偏離 0.5，
> 切勿直接套閉式卡方。
