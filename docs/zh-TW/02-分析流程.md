# 02 · 分析流程：完整六步驟

本章示範如何把[第一章](./01-原理.md)的理論落實成可執行的六步驟。每一步都
對應到 `emdsig/` 內的具體函式。

## 步驟 1：EMD 或 CEEMDAN 分解

把原始時間序列 $x(t)$ 分解成 $n$ 個 IMF $c_m(t)$ 加一個殘差 $r(t)$。

```python
import numpy as np
from PyEMD import CEEMDAN            # 建議用 CEEMDAN 避免模態混疊

x = np.loadtxt("examples/sample_signal.csv")
imfs = CEEMDAN().ceemdan(x)          # shape: (n_imfs, N)
```

> **為什麼推薦 CEEMDAN？** 傳統 EMD 容易產生模態混疊，會讓單一 IMF 的
> $E_m$、$T_m$ 失去物理意義。詳見 [05 避坑指南](./05-避坑指南.md) 第三節。

## 步驟 2：計算每個 IMF 的能量密度 $E_m$

能量密度定義為序列平方和除以長度：

$$E_m = \frac{1}{N} \sum_{i=1}^{N} \big[c_m(t_i)\big]^2$$

本質上就是變異數。`emdsig.metrics.energy` 預設會先裁切兩端各 5%（防禦
邊界效應）再計算。

```python
from emdsig import energy, compute_et
E = energy(imfs[0], trim_ratio=0.05)
# 或一次處理所有 IMF
E_all, T_all = compute_et(imfs, trim_ratio=0.05)
```

## 步驟 3：計算每個 IMF 的平均週期 $T_m$

最簡單穩健的估算方式是**穿零點法**（zero-crossings）：

$$T_m = \frac{N}{N_z}, \quad N_z = \#\{i : \text{sign}\,c_m(t_i) \neq \text{sign}\,c_m(t_{i+1})\}$$

```python
from emdsig import period
T = period(imfs[0])
```

殘差項沒有穿零點，函式會回傳 `NaN`，由上游用 `~np.isnan(T)` 過濾。

## 步驟 4：基準線定錨

Wu & Huang 的公式 $\ln(E) = -\ln(T) + c$ 中的常數 $c$ 需要由一個**可信雜訊 IMF**
來定錨。慣例是用第一個 IMF $c_1$（通常是最高頻，最接近雜訊）：

```python
from emdsig import calibrate_intercept
ln_T = np.log(T_all[~np.isnan(T_all)])
ln_E = np.log(E_all[~np.isnan(T_all)])
c = calibrate_intercept(ln_T[0], ln_E[0])
```

若最高頻本身就是訊號（例如高頻震動量測），這會導致整條基準線被錯誤地
向上拉高。解法請見 [05 避坑指南](./05-避坑指南.md) 第二節。

## 步驟 5：繪製散佈圖 + 信賴上限

```python
from emdsig import chi2_confidence_bounds, plot_significance

N = len(x)
grid = np.linspace(ln_T.min() - 0.5, ln_T.max() + 0.5, 100)
lo, hi = chi2_confidence_bounds(grid, N=N, alpha=0.05, baseline_intercept=c)

plot_significance(ln_T, ln_E,
                  bounds=(grid, lo, hi),
                  baseline_intercept=c,
                  title="EMD 顯著性檢定")
```

## 步驟 6：判讀與過濾

| 位置 | 判讀 | 處理方式 |
|---|---|---|
| 高於上限 | 統計顯著訊號 | **保留**，用於後續特徵萃取或訊號重建。 |
| 位於信賴帶之內 | 與白雜訊無差異 | **剔除**，或視為背景雜訊。 |
| 低於下限 | 能量異常偏低 | 檢查是否為過度平滑或數據瑕疵。 |

重建過濾後的訊號：

```python
significant = ln_E > hi_at_their_ln_T          # 布林索引
filtered_signal = imfs[significant].sum(axis=0)
```
